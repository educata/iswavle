рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃў рЃБрЃцрЃарЃЮ рЃџрЃљрЃЏрЃљрЃќрЃў рЃЊрЃљ рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃњрЃљрЃЋрЃ«рЃљрЃЊрЃЮрЃЌ.

## рЃарЃљ рЃљрЃарЃўрЃА CSS рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ?

CSS рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃўрЃџрЃћрЃЉрЃў рЃЋрЃфрЃЋрЃљрЃџрЃЮрЃЌ рЃЊрЃарЃЮрЃЌрЃљ рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў.
рЃћрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃцрЃћрЃарЃўрЃА, рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃърЃЮрЃќрЃўрЃфрЃўрЃўрЃА рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў CSS рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ.

## рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ?

1. рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў.
2. рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў (рЃарЃљ рЃЊрЃарЃЮрЃерЃў рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃА, рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃА рЃЊрЃљ рЃљ.рЃе).
3. рЃљрЃдрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃўрЃЊрЃљрЃю рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃљрЃЏрЃЊрЃћ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃАрЃЌрЃЋрЃўрЃА рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃърЃўрЃюрЃћрЃарЃўрЃА (рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА) рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ.

рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.

```html preview
<div class="spinner" role="status"></div>
```

```css preview
div.spinner {
  width: 50px;
  height: 50px;
  border: 5px solid #1890ff;
  border-top-color: transparent;
  border-radius: 50%;
}
```

рЃЏрЃўрЃЋрЃўрЃдрЃћрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЕрЃљрЃарЃЕрЃЮ, рЃарЃЮрЃЏрЃћрЃџрЃќрЃћрЃф рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЏрЃЮрЃЋрЃљрЃарЃњрЃЮрЃЌ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ.

:::info
`role="status"` рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ [рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА](./doc/guides/a11y)
:::

рЃњрЃљрЃЋрЃБрЃгрЃћрЃарЃЮрЃЌ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃф:

```css
div.spinner {
  width: 50px;
  height: 50px;
  border: 5px solid #1890ff;
  border-top-color: transparent;
  border-radius: 50%;
  animation-name: spinner;
}
```

[`animation-name`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-name)-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃБрЃЎрЃЋрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃћрЃЌ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў.
рЃљрЃ«рЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА CSS-рЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ.

рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃАрЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃдрЃАрЃљрЃгрЃћрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ [`@keyframes`](https://developer.mozilla.org/en-US/docs/Web/CSS/@keyframes).
рЃЏрЃљрЃАрЃерЃў рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЕрЃљрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў, рЃЮрЃарЃюрЃљрЃўрЃарЃљрЃЊ:

1. `from` рЃЊрЃљ `to`: рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃў (0%) рЃЊрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃў (100%).
2. рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃБрЃџрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃф рЃњрЃЋрЃАрЃБрЃарЃА 0-рЃЊрЃљрЃю 100-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ.

```css
@keyframes spinner {
  from {
    rotate: 0deg;
  }
  to {
    rotate: 360deg;
  }
}
```

[`rotate`](https://developer.mozilla.org/en-US/docs/Web/CSS/rotate) рЃљрЃарЃўрЃА CSS-рЃўрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃўрЃ«рЃљрЃарЃЮрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў.
рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ [`transform: rotate()`](https://developer.mozilla.org/en-US/docs/Web/CSS/transform-function/rotate) рЃцрЃБрЃЦрЃюрЃфрЃўрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

`transform`-рЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃўрЃАрЃљрЃА рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ `transform` рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃАрЃ«рЃЋрЃљ рЃЏрЃ«рЃарЃўрЃЋ рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ рЃљрЃа рЃљрЃЦрЃЋрЃЌ.

рЃЕрЃЋрЃћрЃюрЃА рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљрЃА рЃБрЃЎрЃЋрЃћ рЃљрЃЦрЃЋрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃф рЃЊрЃљ рЃерЃћрЃАрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃћрЃџрЃў рЃАрЃбрЃўрЃџрЃћрЃЉрЃўрЃф рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃЊрЃарЃЮ.
[`animation-duration`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-duration).

рЃЏрЃљрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ `s` (рЃгрЃљрЃЏрЃћрЃЉрЃерЃў) рЃљрЃю `ms` (рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃћрЃЉрЃерЃў).

:::info
рЃћрЃарЃЌрЃў рЃгрЃљрЃЏрЃў рЃљрЃЌрЃљрЃАрЃў рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃўрЃљ ­ЪцЊ
рЃарЃљрЃф рЃюрЃљрЃЎрЃџрЃћрЃЉрЃўрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮ рЃЏрЃўрЃЌ рЃАрЃгрЃарЃљрЃцрЃўрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљрЃф
:::

  }

рЃЕрЃЋрЃћрЃюрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ 10 рЃгрЃљрЃЏрЃў, рЃЏрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЕрЃћрЃарЃЊрЃћрЃЉрЃљ.

## рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў

CSS-рЃерЃў рЃљрЃАрЃћрЃЋрЃћ рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ рЃџрЃљрЃЏрЃљрЃќрЃў рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљрЃерЃў.

[`animation-iteration-count`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-iteration-count)
рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЌрЃБ рЃарЃљрЃЏрЃЊрЃћрЃюрЃ»рЃћрЃа рЃБрЃюрЃЊрЃљ рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃЊрЃћрЃА рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ.

рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌрЃў рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃю `infinite` (рЃБрЃАрЃљрЃАрЃарЃБрЃџрЃЮрЃЊ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА).

### animation-delay

[`animation-delay`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-delay) рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃарЃљрЃЋрЃА рЃЌрЃБ рЃарЃљ рЃЊрЃарЃЮрЃўрЃА
рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ, рЃћрЃарЃЌрЃњрЃЋрЃљрЃарЃў рЃЊрЃљрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮ.

рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ рЃЊрЃарЃЮрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў (`s` рЃљрЃю `ms`).

### animation-direction

[`animation-direction`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-direction) рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃарЃљрЃЋрЃА рЃЌрЃБ рЃарЃљ
рЃЏрЃўрЃЏрЃљрЃарЃЌрЃџрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ.

рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ рЃЮрЃЌрЃ«рЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ:

- `normal` - рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ (`from`-рЃўрЃЊрЃљрЃю `to`-рЃЏрЃЊрЃћ).
- `reverse` - рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃўрЃА рЃерЃћрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ (`to`-рЃўрЃЊрЃљрЃю `from`-рЃЏрЃЊрЃћ).
- `alternate` - рЃўрЃгрЃДрЃћрЃЉрЃљ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃфрЃўрЃЎрЃџрЃўрЃЌ (`from`-рЃўрЃЊрЃљрЃю `to`-рЃЏрЃЊрЃћ) рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃў рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ (`to`-рЃўрЃЊрЃљрЃю `from`-рЃЏрЃЊрЃћ) рЃЊрЃљ рЃљрЃАрЃћ рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃЋрЃџрЃћрЃЮрЃЉрЃўрЃЌ.
- `alternate-reverse` - рЃЏрЃАрЃњрЃљрЃЋрЃАрЃўрЃљ `alternate` рЃЮрЃдрЃЮрЃюрЃЊ рЃерЃћрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃўрЃЌ.

### animation-timing-function

[`animation-timing-function`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-timing-function) рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃарЃљрЃЋрЃА рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа
рЃБрЃюрЃЊрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃфрЃўрЃЎрЃџрЃўрЃА рЃЊрЃарЃЮрЃА.

рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ [cubic-bezier](https://developer.mozilla.org/en-US/docs/Web/CSS/easing-function/cubic-bezier)-рЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ
`cubic-bazier` рЃљрЃарЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃЏрЃЮрЃўрЃарЃњрЃЮ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃАрЃћрЃЌрЃў рЃцрЃЮрЃарЃЏрЃўрЃЌ, рЃарЃЮрЃњрЃЮрЃарЃўрЃф рЃерЃћрЃю рЃњрЃўрЃюрЃЊрЃљ.
рЃћрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА [Bazier curve](https://developer.mozilla.org/en-US/docs/Glossary/Bezier_curve)-рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ. рЃўрЃА рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџ
рЃЊрЃўрЃќрЃљрЃўрЃюрЃерЃў рЃЏрЃарЃБрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃљрЃбрЃљрЃЊ.

`cubic-bazier`-рЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃцрЃЮрЃарЃЏрЃБрЃџрЃљ:

```css
.some_random_element {
  animation-timing-function: cubic-bazier(x1, y1, x2, y2);
}
```

- `x1` - рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ, рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃў X рЃЎрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃў, рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЕрЃљрЃўрЃгрЃћрЃарЃЮрЃА 0-рЃЊрЃљрЃю 1-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ (`[0, 1]`).
- `y1` - рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ, рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃў Y рЃЎрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃў, рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЕрЃљрЃўрЃгрЃћрЃарЃЮрЃА 0-рЃЊрЃљрЃю 1-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ (`[0, 1]`).
- `x2` - рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ, рЃЏрЃћрЃЮрЃарЃћ рЃгрЃћрЃарЃбрЃўрЃџрЃў X рЃЎрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃў, рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЕрЃљрЃўрЃгрЃћрЃарЃЮрЃА 0-рЃЊрЃљрЃю 1-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ (`[0, 1]`).
- `y2` - рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ, рЃЏрЃћрЃЮрЃарЃћ рЃгрЃћрЃарЃбрЃўрЃџрЃў Y рЃЎрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃў, рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЕрЃљрЃўрЃгрЃћрЃарЃЮрЃА 0-рЃЊрЃљрЃю 1-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ (`[0, 1]`).

CSS-рЃерЃў рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃарЃљрЃЏрЃЮрЃЊрЃћрЃюрЃўрЃЏрЃћ cubic-bazier-рЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў:

| рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ       | рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА                                               | cubic-bazier-рЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ          |

## рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЕрЃљрЃгрЃћрЃарЃљ
