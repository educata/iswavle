рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў HTML-рЃерЃў рЃљрЃю рЃЊрЃљрЃЋрЃАрЃбрЃўрЃџрЃЮрЃЌ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃў.

## рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў

```css
selector::pseudo-element {
  property: value;
}
```

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃАрЃЌрЃЋрЃўрЃА, [`::first-line`](https://developer.mozilla.org/en-US/docs/Web/CSS/::first-line) рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА
рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃўрЃА рЃерЃћрЃАрЃљрЃфрЃЋрЃџрЃћрЃџрЃљрЃЊ.

```html preview
<p>рЃљрЃюрЃњрЃБрЃџрЃљрЃарЃў рЃљрЃарЃўрЃА рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃ»рЃљрЃЋрЃљрЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃў, рЃарЃЮрЃЏрЃџрЃўрЃЌрЃљрЃф рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ, рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃљрЃЊ рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃљрЃЊ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃћрЃарЃЌрЃњрЃЋрЃћрЃарЃЊрЃўрЃљрЃюрЃў рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА (SPA - Single Page Application) рЃерЃћрЃЦрЃЏрЃюрЃљ.</p>
```

```css preview
p::first-line {
  color: blue;
}
```

рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃўрЃЌ [рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃАрЃњрЃљрЃю](./doc/guides/html-css/selector-pseudo-classes), рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА
рЃЮрЃарЃЏрЃљрЃњрЃў рЃЮрЃарЃгрЃћрЃарЃбрЃўрЃџрЃў рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ. рЃЌрЃБрЃЏрЃфрЃљ рЃќрЃЮрЃњрЃўрЃћрЃарЃЌ рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ, рЃћрЃарЃЌрЃў рЃЮрЃарЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:
[`::before`](#before), [`::after`](#after), `::first-line` da [`::first-letter`](#first-letter).

:::info
рЃЎрЃљрЃарЃњрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЮрЃарЃЏрЃљрЃњрЃў рЃЮрЃарЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ (`::`).
:::

## рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў

рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃЏрЃАрЃњрЃљрЃЋрЃАрЃўрЃљ рЃарЃћрЃњрЃБрЃџрЃљрЃарЃБрЃџрЃў HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃљ, рЃўрЃАрЃўрЃюрЃўрЃф рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃЌ
рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃЊрЃћрЃЉрЃўрЃљрЃю рЃДрЃБрЃЌрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃерЃў. рЃЏрЃўрЃАрЃў рЃЊрЃљрЃАрЃбрЃўрЃџрЃЋрЃљ рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃўрЃљ рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА.

### before

[`::before`](https://developer.mozilla.org/en-US/docs/Web/CSS/::before) рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃћрЃЏрЃюрЃбрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЦрЃюрЃћрЃЉрЃљ
рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃерЃЋрЃўрЃџрЃЮрЃЉрЃўрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџ рЃЌрЃћрЃњрЃќрЃћ (рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ).

```html preview
<p>
  API-рЃўрЃА рЃерЃћрЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃљрЃЊ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ
  <a href="https://everrest.educata.dev">EverREST-рЃўрЃА </a> рЃљрЃю <a href="https://dummyjson.com/">dummyjson-рЃўрЃА</a>рЃАрЃћрЃарЃЋрЃћрЃарЃў
</p>
```

```css preview
a::before {
  content: '­ЪћЌ';
  color: #1890ff;
}
```

рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў `<a>` рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃерЃў рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃЊрЃљ рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў `before`, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃф рЃљрЃарЃўрЃА `­ЪћЌ`.

:::info
рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃБрЃџрЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃў. <!-- рЃЊрЃљрЃџрЃўрЃюрЃЎрЃћ рЃЕрЃЋрЃћрЃюрЃў рЃАрЃбрЃљрЃбрЃўрЃљ, рЃарЃЮрЃфрЃљ рЃЊрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ -->
Windows-рЃўрЃА рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃЏрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃў рЃЏрЃљрЃџрЃАрЃљрЃ«рЃЏрЃЮрЃЉрЃў: <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd> рЃљрЃю <kbd>F12</kbd>
рЃ«рЃЮрЃџрЃЮ macOS-рЃўрЃА рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃЏрЃљ: <kbd>Рїў</kbd> + <kbd>РїЦ</kbd> + <kbd>I</kbd>.
рЃЊрЃљрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЌ `<a>` рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў.
:::

### after

[`::after`](https://developer.mozilla.org/en-US/docs/Web/CSS/::after) рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЏрЃАрЃњрЃљрЃЋрЃАрЃўрЃљ [`::before`](#before)-рЃўрЃА рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў
рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЉрЃЮрЃџрЃЮ.

```html preview
<p>TODO рЃАрЃўрЃљ:</p>
<ul>
  <li class="done">рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ HTML</li>
  <li class="done">рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ CSS</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ Git</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ JS</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ TS</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ Angular</li>
</ul>
```


### marker

[`::marker`](https://developer.mozilla.org/en-US/docs/Web/CSS/::marker) рЃцрЃАрЃћрЃЋрЃЊрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃўрЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃЌрЃљрЃю, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф
рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА `<li>` рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃА.

рЃЏрЃЮрЃЊрЃўрЃЌ рЃњрЃљрЃЋрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃЮрЃЌ рЃгрЃўрЃюрЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃў:

```html preview
<p>TODO рЃАрЃўрЃљ:</p>
<ul>
  <li class="done">рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ HTML</li>
  <li class="done">рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ CSS</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ Git</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ JS</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ TS</li>
  <li>рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ Angular</li>
</ul>
```

```css preview

}

  content: 'РюЁ';
}
```
