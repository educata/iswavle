[`cookie`](https://developer.mozilla.org/en-US/docs/Web/HTTP/Cookies) рЃљрЃарЃўрЃА рЃЏрЃфрЃўрЃарЃћ рЃќрЃЮрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф
рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃарЃЮрЃА. рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ: рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃА cookie, рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃљрЃ«рЃљрЃџрЃў cookie, рЃњрЃљрЃюрЃљрЃ«рЃџрЃЮрЃА
cookie рЃљрЃю рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊрЃљрЃф рЃгрЃљрЃерЃљрЃџрЃЮрЃА рЃўрЃА.

## рЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ cookie

**cookie**-рЃўрЃА рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ рЃЏрЃћрЃбрЃгрЃўрЃџрЃљрЃЊрЃљрЃЊ рЃљрЃарЃўрЃА рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃю рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА
рЃгрЃљрЃарЃАрЃљрЃЊрЃњрЃћрЃюрЃљрЃЊ. рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ рЃАрЃЮрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃЦрЃАрЃћрЃџрЃўрЃА рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃў, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃћрЃџрЃЏрЃљ рЃњрЃљрЃўрЃљрЃарЃљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃўрЃ«рЃўрЃџрЃљ рЃЏрЃўрЃАрЃў
рЃАрЃўрЃљрЃ«рЃџрЃћрЃћрЃЉрЃў. рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃЊрЃарЃЮрЃўрЃА рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏ рЃЎрЃў рЃњрЃљрЃЌрЃўрЃерЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў рЃљрЃю рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊрЃљрЃф рЃЊрЃљрЃљрЃарЃћрЃцрЃарЃћрЃерЃљ (рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃљ) рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ. рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў
рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃЏрЃљрЃА рЃЋрЃћрЃдрЃљрЃа рЃфрЃюрЃЮрЃЉрЃА. рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЏрЃўрЃАрЃў рЃЎрЃўрЃЊрЃћрЃЋ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ.
рЃДрЃЮрЃЋрЃћрЃџ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃќрЃћ рЃарЃЮрЃЏ рЃџрЃЮрЃњрЃўрЃюрЃўрЃА рЃЊрЃљ рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃљ рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃЮрЃЊрЃћрЃА, рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃАрЃљрЃерЃўрЃюрЃџрЃљрЃЊ рЃЊрЃљрЃўрЃбрЃљрЃюрЃ»рЃћрЃЉрЃЮрЃЊрЃљ.
рЃАрЃљрЃюрЃљрЃфрЃЋрЃџрЃЮрЃЊ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ cookie, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃерЃћрЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃ«рЃћрЃЉ рЃЏрЃфрЃўрЃарЃћ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА.

рЃљрЃЏрЃарЃўрЃњрЃљрЃЊ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏ, рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃфрЃюрЃЮрЃЉрЃА рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃЊрЃљ рЃБрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃљрЃўрЃА cookie-рЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃЏрЃЊрЃљрЃўрЃюрЃарЃћ.

Cookie-рЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃљрЃЊ рЃАрЃљрЃЏрЃў рЃЏрЃўрЃќрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА:

- **рЃАрЃћрЃАрЃўрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ** - рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃА рЃЏрЃЮрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљрЃерЃў.
- **рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ** - рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃћрЃџрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃћрЃюрЃљ, рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃАрЃбрЃўрЃџрЃў рЃЊрЃљ рЃљ.рЃе.
- **Tracking** - рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЦрЃфрЃћрЃЋрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЊрЃљрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃћрЃЉрЃљ.

## рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ

рЃБрЃгрЃўрЃю рЃљрЃа рЃўрЃДрЃЮ рЃЋрЃћрЃЉрЃерЃў cookie-рЃА рЃњрЃљрЃарЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃЋрЃљ, рЃарЃЮрЃЏ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃерЃћрЃњрЃЋрЃћрЃюрЃљрЃ«рЃљ. рЃЌрЃБрЃЏрЃфрЃљ рЃЊрЃдрЃћрЃАрЃЊрЃдрЃћрЃўрЃАрЃЮрЃЉрЃўрЃЌ рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃЉрЃћрЃЋрЃарЃў рЃЎрЃљрЃарЃњрЃў рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃЋрЃљ,

рЃЉрЃћрЃЋрЃарЃў Cookie-рЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљрЃф рЃљрЃа рЃљрЃарЃўрЃА рЃЎрЃљрЃарЃњрЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃЦрЃЋрЃА рЃюрЃћрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃў,
рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃљрЃЏ cookie-рЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљрЃЏ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЊрЃарЃЮрЃА рЃњрЃљрЃќрЃљрЃарЃЊрЃЮрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃЊрЃарЃЮ.

## Cookie-рЃќрЃћ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў

Cookie-рЃА рЃЏрЃћрЃбрЃгрЃўрЃџрЃљрЃЊрЃљрЃЊ рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃљрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃА рЃЊрЃљ рЃњрЃЋрЃўрЃњрЃќрЃљрЃЋрЃюрЃўрЃА. рЃЌрЃБрЃЏрЃфрЃљ рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃАрЃљрЃф рЃљрЃарЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃў cookie-рЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.

рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ [`document.cookie`](https://developer.mozilla.org/en-US/docs/Web/API/Document/cookie), рЃАрЃљрЃЊрЃљрЃф рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў cookie-рЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ.

```js
document.cookie = 'name=Educata';
document.cookie = 'api=EverREST';
```

рЃљрЃЏ рЃЎрЃЮрЃЊрЃўрЃЌ рЃерЃћрЃЋрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЮрЃарЃў cookie, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљрЃф рЃљрЃарЃўрЃА `name` рЃЊрЃљ `api`, рЃ«рЃЮрЃџрЃЮ рЃЏрЃљрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЎрЃў - `Educata` рЃЊрЃљ `EverREST`.

Cookie рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃўрЃ«рЃўрЃџрЃЮрЃЌ developer tools-рЃерЃў: inspect -> application -> cookie

![Cookie-рЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў](./assets/images/cookie.png)

рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ cookie-рЃА рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃў рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў:

- **Name** - Cookie-рЃА рЃЊрЃљрЃАрЃљрЃ«рЃџрЃћрЃЉрЃљ. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ cookie-рЃА рЃБрЃюрЃЊрЃљ рЃњрЃљрЃљрЃЕрЃюрЃЊрЃћрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў, рЃАрЃ«рЃЋрЃљ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃљрЃ«рЃџрЃљрЃЊ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃљрЃгрЃћрЃарЃА рЃгрЃўрЃюрЃљ cookie-рЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃА.
- **Value** - Cookie-рЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.
- **Domain** - рЃарЃЮрЃЏрЃћрЃџрЃў рЃЊрЃЮрЃЏрЃћрЃўрЃюрЃА рЃљрЃЦрЃЋрЃА рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃљ. рЃЌрЃБ рЃљрЃарЃљрЃцрЃћрЃарЃў рЃљрЃа рЃљрЃарЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЏрЃљрЃерЃўрЃю cookie-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃА, рЃАрЃљрЃЊрЃљрЃф рЃўрЃА рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ. рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃљрЃЉрЃЊрЃЮрЃЏрЃћрЃўрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃљрЃф, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: `.Domain=.iswavle.com` рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА cookie-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ рЃЊрЃЮрЃЏрЃћрЃўрЃюрЃќрЃћ рЃАрЃљрЃЊрЃљрЃф рЃерЃћрЃЦрЃўрЃЏрЃюрЃљ рЃЊрЃљ рЃљрЃАрЃћрЃЋрЃћ рЃАрЃљрЃЉрЃЊрЃЮрЃЏрЃћрЃўрЃюрЃћрЃЉрЃќрЃћрЃф. рЃЕрЃЋрЃћрЃюрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃЌрЃБ cookie рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ `iswavle.com`-рЃќрЃћ рЃЏрЃљрЃерЃўрЃю рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф `iswavle.com`-рЃќрЃћ рЃљрЃАрЃћрЃЋрЃћ `practice.iswavle.com`.
- **Path** - рЃарЃЮрЃЏрЃћрЃџрЃў URL-рЃАрЃЌрЃЋрЃўрЃА рЃљрЃарЃўрЃА рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃюрЃћрЃЉрЃљрЃЊрЃљрЃарЃЌрЃБрЃџрЃў. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃЌрЃБ рЃљрЃарЃўрЃА `/` рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃљрЃЊрЃњрЃўрЃџрЃўрЃЊрЃљрЃю рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.
- **Expires** / Max-Age - рЃарЃЮрЃЊрЃўрЃА рЃгрЃљрЃўрЃерЃљрЃџрЃЮрЃА cookie. рЃЌрЃБ рЃљрЃа рЃЏрЃўрЃЋрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃЌ рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃљрЃЊ рЃЏрЃўрЃћрЃюрЃўрЃГрЃћрЃЉрЃљ рЃАрЃћрЃАрЃўрЃБрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.
  - **Expires** - рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЌрЃљрЃарЃўрЃдрЃА. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: `Expires=Tue, 22 Jul 2025 12:00:00 UTC` рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃарЃЮрЃЏ cookie-рЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃўрЃљ 2025 рЃгрЃџрЃўрЃА 22 рЃўрЃЋрЃџрЃўрЃАрЃљрЃЏрЃЊрЃћ.
  - **Max**-Age - рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃЊрЃљрЃю рЃарЃљ рЃЊрЃарЃЮрЃерЃў рЃгрЃљрЃўрЃерЃљрЃџрЃЮрЃА. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: `Max-Age=3600` рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА, рЃарЃЮрЃЏ cookie рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃўрЃљ 1 рЃАрЃљрЃљрЃЌрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў (1 рЃАрЃљрЃљрЃЌрЃў = 3600 рЃгрЃљрЃЏрЃА).
- **Size** - Cookie-рЃА рЃќрЃЮрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃўрЃљ рЃЉрЃљрЃўрЃбрЃћрЃЉрЃерЃў. рЃБрЃЏрЃћрЃбрЃћрЃАрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА 4096 рЃЉрЃљрЃўрЃбрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃќрЃЮрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃљрЃф рЃљрЃарЃўрЃА 4 рЃЎрЃўрЃџрЃЮрЃЉрЃљрЃўрЃбрЃў. рЃЌрЃБ рЃљрЃЏ рЃќрЃЮрЃЏрЃљрЃА рЃњрЃљрЃЊрЃљрЃфрЃЊрЃћрЃЉрЃўрЃЌ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ cookie рЃљрЃа рЃЊрЃљрЃДрЃћрЃюрЃЊрЃћрЃА.
- **HttpOnly** - рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃА `document.cookie`-рЃАрЃЌрЃљрЃю рЃЏрЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљрЃерЃў. рЃЌрЃБ `HttpOnly` рЃљрЃарЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЉрЃљ, рЃарЃЮрЃЏ рЃЎрЃџрЃўрЃћрЃюрЃбрЃА рЃљрЃа рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ JavaScript-рЃўрЃЌ. рЃћрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ рЃЎрЃљрЃарЃњрЃўрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА рЃЎрЃЮрЃЊрЃЏрЃљ, рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃа рЃЏрЃўрЃўрЃдрЃЮрЃА cookie-рЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ.
- **Secure** - рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЊрЃЮрЃюрЃћрЃА рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЌрЃБ Secure рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ cookie-рЃерЃў, рЃўрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЉрЃљ, рЃарЃЮрЃЏ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЊрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ HTTPS-рЃўрЃЌ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ рЃљрЃЏ cookie-рЃќрЃћ.
- **SameSite** - рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃА cross-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА. рЃљрЃЏ рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЌрЃљрЃЋрЃў рЃљрЃЋрЃљрЃарЃўрЃЊрЃЮрЃЌ Cross-Site Request Forgery-рЃА, рЃўрЃњрЃўрЃЋрЃћ CSRF-рЃА рЃерЃћрЃбрЃћрЃЋрЃћрЃЉрЃА. рЃЏрЃљрЃА рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ рЃАрЃљрЃЏрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ:
  - `SameSite=Stict` - cookie рЃљрЃа рЃњрЃљрЃњрЃўрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ cross-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЊрЃарЃЮрЃА.
  - `SameSite=Lax` - cookie рЃњрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ same-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА рЃЊрЃљ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ cross-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЊрЃарЃЮрЃА (GET рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃЊрЃарЃЮрЃА).
  - `SameSite=None` - cookie рЃњрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ cross-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЊрЃарЃЮрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃарЃЮрЃЏ рЃўрЃЏрЃБрЃерЃљрЃЋрЃЮрЃА, рЃАрЃљрЃГрЃўрЃарЃЮрЃљ Secure рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў рЃўрЃДрЃЮрЃА.
- **Partition Key Site** - [Partitioned (рЃЊрЃљрЃюрЃљрЃгрЃћрЃЋрЃарЃћрЃЉрЃБрЃџрЃў) cookie](https://developer.mozilla.org/en-US/docs/Web/Privacy/Privacy_sandbox/Partitioned_cookies) рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃўрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ cross-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА рЃљрЃа рЃЏрЃЮрЃ«рЃЊрЃћрЃА cookie-рЃќрЃћ рЃЌрЃЋрЃљрЃџрЃЌрЃЋрЃљрЃџрЃў (tracking).
- **Cross Site** - рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃЏрЃЮрЃўрЃЦрЃфрЃћрЃА cross-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА.
- **Priority** - Cookie-рЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃБрЃџрЃЮрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ 3 рЃЊрЃЮрЃюрЃћ:
  - `Priority=Low` - рЃЊрЃљрЃЉрЃљрЃџрЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃБрЃџрЃЮрЃЉрЃљ, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЊрЃарЃЮрЃАрЃљрЃф cookie рЃерЃћрЃАрЃљрЃФрЃџрЃЮрЃљ рЃгрЃљрЃўрЃерЃљрЃџрЃЮрЃА рЃЌрЃБ рЃЏрЃўрЃљрЃдрЃгрЃўрЃљ рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃа рЃбрЃћрЃЋрЃљрЃЊрЃЮрЃЉрЃљрЃА.
  - `Priority=Medium` - рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў (default) рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃБрЃџрЃЮрЃЉрЃљ, рЃљрЃарЃљрЃцрЃћрЃарЃў рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃБрЃџрЃў.
  - `Priorty=High` - рЃЏрЃљрЃдрЃљрЃџрЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃБрЃџрЃЮрЃЉрЃљ, рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃБрЃюрЃЊрЃљ рЃерЃћрЃћрЃфрЃљрЃЊрЃЮрЃА рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃЏрЃљ, рЃарЃЮрЃЏ рЃћрЃА cookie рЃљрЃа рЃгрЃљрЃерЃљрЃџрЃЮрЃА.

:::info
cross-site рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ рЃљрЃарЃўрЃА рЃўрЃАрЃћрЃЌрЃў рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ, рЃарЃЮрЃфрЃљ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃбрЃўрЃърЃўрЃА рЃЊрЃЮрЃЏрЃћрЃўрЃюрЃћрЃЉрЃў рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА.
:::

рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ cookie, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃДрЃЮрЃЋрЃћрЃџрЃўрЃЋрЃћ рЃљрЃЏ рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃА:

```js
document.cookie = 'sessionID=123456789abc; Max-Age=3600; domain=.iswavle.com; path=/; HttpOnly; Secure; SameSite=Lax; Priority=High; Partitioned';
```

рЃарЃљрЃЊрЃњрЃљрЃю `HttpOnly` рЃЊрЃљрЃЋрЃБрЃгрЃћрЃарЃћрЃЌ, cookie рЃљрЃа рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃА рЃЌрЃБрЃџрЃАрЃерЃў (Application) рЃЌрЃБрЃЏрЃфрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЊрЃарЃЮрЃА рЃњрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю. рЃЌрЃБ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃўрЃЌ
`HttpOnly`-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, рЃЏрЃљрЃерЃўрЃю рЃЏрЃљрЃА рЃерЃћрЃЋрЃљрЃЏрЃЕрЃюрЃћрЃЋрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЊрЃљрЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљрЃф рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ. рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ `HttpOnly` рЃБрЃАрЃљрЃцрЃарЃЌрЃЮрЃ«рЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## CRUD рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў cookie-рЃќрЃћ

рЃЉрЃћрЃЋрЃарЃў рЃЊрЃљрЃЏрЃ«рЃљрЃЏрЃарЃћ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА cookie-рЃАрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃњрЃЋрЃўрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА CRUD-рЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃА.
рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃЌрЃБ рЃБрЃцрЃарЃЮ рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃћрЃЉрЃў рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ, рЃЏрЃљрЃерЃўрЃю `document.cookie`-рЃА рЃгрЃћрЃарЃљ рЃЏрЃЮрЃБрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃўрЃљ.
рЃљрЃЏрЃўрЃА рЃЏрЃЮрЃАрЃљрЃњрЃЋрЃљрЃарЃћрЃЉрЃџрЃљрЃЊ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃћрЃЉрЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃљрЃЋрЃћрЃю рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃА, рЃљрЃю рЃЌрЃљрЃЋрЃљрЃЊ рЃљрЃЊрЃњрЃћрЃюрЃћрЃю cookie-рЃАрЃЌрЃљрЃю рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃћрЃЉрЃА.

рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃЮрЃЌ рЃљрЃАрЃћрЃЌрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

```js
class CookieService {
  static setCookie(name, value, options = {}) {
    let cookieString = `${encodeURIComponent(name)}=${encodeURIComponent(value)}`;

    if (options.expires) {
      const expires = typeof options.expires === 'number' ? new Date(Date.now() + options.expires * 864e5).toUTCString() : options.expires.toUTCString();
      cookieString += `; expires=${expires}`;
    }

    if (options.path) {
      cookieString += `; path=${options.path}`;
    }

    if (options.domain) {
      cookieString += `; domain=${options.domain}`;
    }

    if (options.secure) {
      cookieString += '; secure';
    }

    if (options.httpOnly) {
      cookieString += '; HttpOnly';
    }

    if (options.sameSite) {
      cookieString += `; SameSite=${options.sameSite}`;
    }

    document.cookie = cookieString;
  }

  static getCookie(name) {
    const cookieMatch = document.cookie.match(new RegExp('(^| )' + encodeURIComponent(name) + '=([^;]+)'));
    return cookieMatch ? decodeURIComponent(cookieMatch[2]) : null;
  }

  static updateCookie(name, value, options) {
    this.setCookie(name, value, options);
  }

  static deleteCookie(name, options = {}) {
    this.setCookie(name, '', { expires: -1, ...options });
  }

  static listCookies() {
    return document.cookie.split('; ').map((cookie) => {
      const [name, value] = cookie.split('=');
      return { name: decodeURIComponent(name), value: decodeURIComponent(value) };
    });
  }

  static clear() {
    const cookies = document.cookie.split('; ');
    for (const cookie of cookies) {
      const name = cookie.split('=')[0];
      this.deleteCookie(name);
    }
  }
}

// рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
CookieService.setCookie('test', 'value', { expires: 7, path: '/', secure: true, sameSite: 'Lax' }); // cookie-рЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
console.log(CookieService.getCookie('test')); // cookie-рЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
CookieService.updateCookie('test', 'newValue', { expires: 7 }); // cookie-рЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
console.log(CookieService.getCookie('test')); // рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў cookie-рЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
CookieService.deleteCookie('test', { path: '/' }); // cookie-рЃА рЃгрЃљрЃерЃџрЃљ
console.log(CookieService.getCookie('test')); // рЃгрЃљрЃерЃџрЃўрЃџрЃў cookie-рЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
console.log(CookieService.listCookies()); // рЃДрЃЮрЃЋрЃћрЃџрЃў cookie-рЃА рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ
CookieService.clear(); // рЃДрЃЮрЃЋрЃћрЃџрЃў cookie-рЃА рЃгрЃљрЃерЃџрЃљ
```

рЃљрЃЏрЃарЃўрЃњрЃљрЃЊ рЃЏрЃўрЃЋрЃўрЃдрЃћрЃЌ рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ cookie-рЃќрЃћ CRUD-рЃўрЃА рЃбрЃўрЃърЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃџрЃљрЃЊ.
рЃарЃљрЃЊрЃњрЃљрЃюрЃљрЃф рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў рЃљрЃарЃўрЃА `static`-рЃўрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў, рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЎрЃџрЃљрЃАрЃўрЃА рЃљрЃ«рЃљрЃџрЃў рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ `new` рЃЦрЃўрЃЋрЃЮрЃарЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

## рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

Cookie рЃћрЃА рЃљрЃарЃўрЃА рЃћрЃарЃЌрЃњрЃЋрЃљрЃарЃў рЃАрЃљрЃфрЃљрЃЋрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў cookie рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЋрЃљрЃДрЃЮрЃџрЃЮрЃЌ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃА.
рЃЌрЃБ cookie-рЃА рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ `HttpOnly` рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ, рЃЏрЃљрЃерЃўрЃю рЃўрЃА рЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃ«рЃўрЃџрЃЋрЃљрЃЊрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА. Cookie рЃљрЃарЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃърЃљрЃбрЃљрЃарЃљ рЃАрЃљрЃфрЃљрЃЋрЃў, рЃљрЃЏрЃўрЃбрЃЮрЃЏрЃљрЃф рЃЏрЃљрЃАрЃерЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃљрЃЊ
рЃАрЃљрЃЏрЃў рЃбрЃўрЃърЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ: рЃАрЃћрЃАрЃўрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ, рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃАрЃљрЃЌрЃЋрЃљрЃџрЃЌрЃЋрЃљрЃџрЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ.
